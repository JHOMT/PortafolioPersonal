Proyectos

En esta sección, te presento algunos de los proyectos en los que he trabajado, demostrando mi pasión por la ingeniería en sistemas e informática y mi habilidad para aplicar conocimientos en tecnologías actuales. Cada proyecto representa un desafío único y una oportunidad para aprender y crecer como profesional en el campo de la programación y el desarrollo de software.

Proyectos Frontend

MusicMixWeb: Un reproductor de música en línea donde los usuarios pueden explorar, buscar y reproducir canciones de diferentes géneros. El proyecto se construyó utilizando HTML, CSS y JavaScript para la interfaz de usuario.

Encriptador de Texto: Una aplicación web que permite a los usuarios encriptar y desencriptar mensajes de texto utilizando diferentes algoritmos de cifrado. El proyecto se desarrolló utilizando HTML, CSS y JavaScript para la funcionalidad de encriptación, fue parte de un challege del programa One Oracle Next Education.

Barberia Alura: Un sitio web para una barbería ficticia donde los usuarios pueden ver los servicios ofrecidos, hacer reservas y obtener información sobre el negocio. El proyecto se implementó utilizando HTML, CSS y JavaScript para la interacción con el usuario.

Proyectos Backend

Conversor: Un proyecto de conversión de unidades donde los usuarios pueden convertir diferentes tipos de unidades, como longitud, temperatura, etc. El proyecto se implementó en el lado del servidor utilizando tecnologías como Java, fue parte de un challege del programa One Oracle Next Education.

Snake: Una versión del clásico juego Snake, donde los jugadores controlan una serpiente y deben comer la mayor cantidad de alimentos posibles sin chocar contra las paredes o el propio cuerpo. El proyecto se desarrolló utilizando Java.

Proyecto "Dibujador de Figuras Geométricas": Una aplicación en Java con interfaz gráfica que permite a los usuarios dibujar círculos, triángulos y cuadrados. Incluye opciones de personalización como tamaño. Además, cuenta con una funcionalidad de inicio de sesión.
